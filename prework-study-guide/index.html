HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements can contain elements inside, called the child element.
  • Elements that contain child elements are called parent elements.
  • An HTML attribute defines addional information about an element.

CSS

  • A margin indicates how much space we want around the outside of the element
  • A padding indicates how much space we want around the content inside an element
  • An external CSS stylesheet is a separate code that lists the CSS style rules. It must be linked to the HTML code
  • Class attributes allow us to share a CSS rule to an element we choose by assigning the rule to a class attribute with a class selector.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and swithces to it
  • main branch: the main branch where our work will end up when it is done.
  • feature branch: when we make a new branch to work on, it is called the feature branch.
  • git pull origin main: pulls the most recent up to date main branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An array is a varaible that contains a group of data.
  • A function is a set of instructions that tells the computer how to perform a task.
  • A conditional statement will only run code if the condition is true.
